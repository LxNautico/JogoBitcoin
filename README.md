# 🪙 JogoBitcoin — Simulador Educacional de Criptografia e Blockchain

Um jogo interativo que ensina conceitos de criptografia, hashes, blockchain, prova de trabalho e transações, tudo de forma prática e gamificada.  
Ideal para estudantes, curiosos e desenvolvedores que querem aprender brincando.

## 🚀 Funcionalidades

- Desafios com criptografia real  
- Progresso por blocos simulando mineração  
- Interface simples e intuitiva  
- Feedback dinâmico ao usuário  
- Hospedagem online para jogar direto no navegador  
- Código aberto para estudo e colaboração
